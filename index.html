Esteganografía LSB

Proyecto final

Alejandro Résendiz Martinez
Marcos Zárraga Flores

Este proyecto nos muestra el algoritmo de esteganografía LSB (Least significant bit), donde se lee cada pixel de la imagen y se desglosa en bits, una vez teniendo esta información, se altera el ultimo bit de la secuencia de colores para poder ocultar el mensaje en la imagen y que el ojo humano no pueda diferenciar a simple vista las modificaciones realizadas a la imagen original.
